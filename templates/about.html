About MedoChat

More than a messaging platform.

Our Story

MedoChat started as a personal project with one simple goal: create a modern messaging platform from scratch using HTML, CSS, JavaScript and Python.

Every page, animation, feature and backend component is being built step by step to understand how real-world chat applications work.

Our Mission

To build a communication platform that is simple, fast, secure and enjoyable to use while continuously learning modern web development.

Current Features

  • Modern User Interface
  • Secure Login & Registration
  • Real-Time Messaging Interface
  • AI Assistant Tools
  • Profile Customization
  • Responsive Design

Future Roadmap

  • Private Messaging
  • Group Chats
  • Voice Messages
  • Video Calling
  • File Sharing
  • AI Chat Assistant
  • TravelSync Integration
